jenkins安装教程


工具&原料

jdk

tomcat

jenkins

centos

详细信息如下：

[root@jenkins100 iso]# cat /etc/redhat-release

CentOS Linux release 7.4.1708 (Core)

[root@jenkins100 iso]# ls

apache-tomcat-8.5.42.tar.gz  jdk-8u211-linux-x64.tar.gz  jenkins.war


方法&步骤

1、jdk安装

[root@jenkins100 iso]# tar -xvf jdk-8u211-linux-x64.tar.gz -C /usr/local/

[root@jenkins100 iso]# export 'PATH=/usr/local/jdk1.8.0_211/bin/:$PATH' >> /etc/profile

[root@jenkins100 iso]# source /etc/profile

[root@jenkins100 iso]# java -version

java version "1.8.0_211"

Java(TM) SE Runtime Environment (build 1.8.0_211-b12)

Java HotSpot(TM) 64-Bit Server VM (build 25.211-b12, mixed mode)

2、tomcat安装

[root@jenkins100 iso]# tar -xvf apache-tomcat-8.5.42.tar.gz

[root@jenkins100 iso]# mv apache-tomcat-8.5.42 /usr/local/tomcat

[root@jenkins100 iso]# /usr/local/tomcat/bin/startup.sh

[root@jenkins100 iso]# ss -tunlp| grep :8080

tcp    LISTEN    0      100      :::8080                :::*                  users:(("java",pid=11783,fd=49))

3、jenkins安装

[root@jenkins100 iso]# cp jenkins.war /usr/local/tomcat/webapps/

在浏览器输入http://自己ip:8080/jenkins，具体操作如下图








4、配置tomcat

[root@jenkins100 ~]# vim /usr/local/tomcat/conf/server.xml

.......

<Host name="localhost" appBase="webapps"

            unpackWARs="true" autoDeploy="true">

      <Context path="" docBase="jenkins" reloadable="flase" />

........

[root@jenkins100 ~]# /usr/local/tomcat/bin/startup.sh

现在可直接访问http://ip:8080






总结

jenkins安装比较简单，安装方法有很多种，本次只是分享其中一种安装方式；安装步骤到第三步就完成了，第四步可做可不做，我建议配置一下访问比较方便。


温馨提示

jenkins的安装也可直接运行：java jar jenkins.war，然后访问http://ip:8080
